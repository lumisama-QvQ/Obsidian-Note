# zsh历史记录清理
zsh的`zhistory`中会记录不少脏乱错误的命令，而若在`zshrc`中开启了`HIST_IGNORE_SPACE`该功能

则可以在不想记录在历史中的命令前加上一个**空格**来让zsh忽略记录

当然可以用`atuin`来代替`zsh`的历史功能
在`zshrc`中添加一下内容：
```bash
# 禁止 Atuin 接管“上箭头”，只接管 Ctrl+R
export ATUIN_NOBIND="true" 
eval "$(atuin init zsh)"
# 手动绑定 Ctrl+R 触发 Atuin
bindkey '^r' _atuin_search_widget
```

安装`atuin`后执行`atuin import zsh`导入历史记录，之后就可以删除`zhistory`啦(・∀・)

# 使用`zellij`来丰富终端体验

`zellij`是由`rust`编写的新一代终端复用器，相比与`tmux`的逆天快捷键，它更易上手，得益于继承了`rust`优良的传统，它的[文档](https://zellij.dev/documentation/)相当的详细。
推荐[阅读](https://sspai.com/prime/story/get-started-with-zellij)，里面有更为详尽的介绍。

`zellij`默认的界面就提供有不错的快捷键指引，这里提一些技巧
- `alt`+`f`可调出一个浮动窗口，可用来临时执行一些命令。
- `alt`+`n`可方便的新建窗格
- `alt`+`方向键`可在窗格之间切换
- `alt`+`/`可在默认布局中切换4
- `zellij`会默认把鼠标选中的内容复制到系统粘贴板里，可在默认的设置文件里通过`copy_on_select`关键词开关
- 可通过在`zellij`后加上参数`attach -c MY_ZELLIJ_SESSION`来使其启动时连接到会话`MY_ZELLIJ_SESSION`,而在第一次启动时自动创建会话
- `alt` + `-`/`+`可快速横向调节窗格大小
