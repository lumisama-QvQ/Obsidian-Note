# 解决在`XWayland`下输入法不生效的问题

在`Wayland`下`X11`的软件不会自动沿用相关配置，这就需要我们手动配置相关的**环境变量**
在`/etc/environment`文件中写入：
```bash
XMODIFIERS=@im=fcitx
QT_IM_MODULE=fcitx
SDL_IM_MODULE=fcitx #针对steam
#GTK_IM_MODULE=fcitx #不建议开启，会影响Wayland下软件的输入
```

# 解决`sway`下使用`AUR`包管理的`bar`信息同步问题
#sway 
在`Manjaro`支持的`sway`版中默认的更新管理插件通讯会指定为执行环境用户，这在使用`pacman`时没事，但在遇到使用`AUR`包管理这种中途提权的进程会导致通讯错误，作为桌面用户的我们只需删除这相关参数即可。
```bash 
#/usr/bin/waybar-signal
...

pkill -x  -SIGRTMIN+${number} 'waybar'
```

# 终端查看文件占用和大小
`dua-cli`是一个快速的磁盘分析器
可在命令行敲入`dua`快速的按降序，易读的展示当前目录的各文件大小
也可敲入`dua i`进入交互模式，友好、现代的的查看文件大小分析
