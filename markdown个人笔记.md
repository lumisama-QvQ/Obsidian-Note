# 文本格式

- 单`*`或`_`斜体
-  双`*`或`_`粗体
- 在一行仅使用三个以上的`*`,`_`或`-`来创建分隔线
- 双`~`删除线
- 下划线需使用`<u>`和`</u>`包围
- `` ` ``包围需要标记的代码
>当标记的代码中包含`` ` ``用双`` ` ``来包含
- 双`=`包含需要高亮的字体
- 脚注使用`[^要脚注的文本]`，随后在适当位置`[要脚注的文本]:脚注内容`
- 使用多个空行来创建段落
>如需在不创建段落下换行，请在换行前加上2个以上的空格
- `<br>`可用来表示换行

# 列表

在行首加：
- `-`/`+`/`*`无序列表
- 数字+`.`有序列表
- `- [ ]`未完成的任务列表
- `- [x]`已完成的任务列表

# 数学公式

在一行内使用：用`$`包裹数学公式，而块则使用双`$`
常用速记：
多行使用`align`创建:
	`$$`
	`\begin{align}`
	`...`
	`\end{align}`
	`$$`

| `\sum`, `\prod`, `\int`      | 求和、乘积、积分                 |
| ---------------------------- | ------------------------ |
| `\frac{分子}{分母}`              | 分数                       |
| `\sqrt{表达式}`                 | 平方根                      |
| `\sqrt[n]{表达式}`              | n次根                      |
| ``+`, `-`, `\times`, `\div`` | 加减乘除                     |
| `x^2`, `e^{i\pi`             | $x^2$, $e^{i\pi}$        |
| `=`, `\neq`, `\equiv`        | $=$, $\neq$, $\equiv$    |
| `\approx`, `\sim`            | $\approx$, $\sim$        |
| `<`, `>`, `\leq`, `\geq`     | $<$, $>$, $\leq$, $\geq$ |
| `\in`, `\notin`              | $\in$, $\notin$          |
| `\subset`, `\supset`         | $\subset$, $\supset$     |
| `\cap`, `\cup`               | $\cap$, $\cup$           |
| `\emptyset`                  | $\emptyset$              |

## 特殊函数和符号

- 三角函数：`\sin`, `\cos`, `\tan`
- 对数：`\log`, `\ln`
- 极限：`\lim_{x \to 0}`
- 求和：`\sum_{i=1}^{n}`
- 积分：`\int_{a}^{b}`
- 无穷：`\infty`
## 矩阵表示

```
$$
    \begin{pmatrix}
    a & b \\
    c & d
    \end{pmatrix}
    $$
```

不同括号类型的矩阵:
- `pmatrix`：圆括号 $\begin{pmatrix} a & b \ c & d \end{pmatrix}$
- `bmatrix`：方括号 $\begin{bmatrix} a & b \ c & d \end{bmatrix}$
- `vmatrix`：行列式 $\begin{vmatrix} a & b \ c & d \end{vmatrix}$
