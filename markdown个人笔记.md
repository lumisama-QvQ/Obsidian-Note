# 标题

不同数量的`#`表示不同等级的标题
还可以创建锚点来跳转到指定的标题`[显示的内容](#+要跳转的标题名称)
例如：[跳转](#标题)
`#`和标题内容之间要有空格，不然会变成标签  

# 文本格式

- 单`*`或`_`斜体
-  双`*`或`_`粗体
- 在一行仅使用三个以上的`*`或`-`来创建分隔线
- 双`~`删除线
- 下划线需使用`<u>`和`</u>`包围
- `` ` ``包围需要标记的代码
>当标记的代码中包含`` ` ``用双`` ` ``来包含
- 双`=`包含需要高亮的字体
- 脚注使用`[^要脚注的文本]`，随后在适当位置`[要脚注的文本]:脚注内容`
- 使用多个空行来创建段落
>如需在不创建段落下换行，请在换行前加上2个以上的空格
